# React Leaflet Web Map Demo


## Install:
Make sure you have Node.JS @5.9.1 or greater installed globally.

Then do: `npm install`

## To Run
Do `npm run start` in the root of this repo, Open a browser at `http://localhost:3000` to see the live app.
